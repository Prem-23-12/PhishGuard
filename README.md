# 🔐 PhishGuard

PhishGuard is a full-stack phishing URL detection tool that helps users identify malicious links using rule-based security analysis.

## 🚀 Features
- Detects phishing URLs
- Validates URL input (SAFE / PHISHING / INVALID)
- Visual risk meter
- Clear explanation of risks
- Keyboard-friendly (Enter to scan)
- Clean, modern UI

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Security: URL parsing, regex-based detection
